# ProLean – Programming Learning Platform

ProLean is a web-based learning platform developed as my final year undergraduate project in 2022.
The platform provides programming tutorials, reference books, and learning resources for students who want to learn programming languages.

## Features

* Programming tutorials for multiple languages
* Programming books and reference materials
* Educational resources for beginners
* Feedback system for user suggestions
* Clean and simple learning interface

## Technologies Used

Frontend

* HTML
* CSS
* JavaScript

Backend / Database

* Firebase (used for storing purchase and transaction data)

Form Handling

* Formspree API (used for collecting user feedback)

## Project Objective

The objective of this project was to create a centralized platform where beginners can easily access programming learning materials, tutorials, and books in one place.

## Folder Structure

* HTML pages for course and book navigation
* CSS files for styling
* JavaScript for interactions
* Image assets and learning resources
* Feedback form integration
