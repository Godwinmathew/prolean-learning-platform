# ProLean – Programming Learning Platform

ProLean is a web-based learning platform developed as my undergraduate final year project (2022). The platform provides programming tutorials, reference books, and learning resources for beginners.

## Features

* Programming tutorials for languages like C, C++, Java, Python, and Kotlin
* Programming books and reference materials
* Structured learning interface
* Feedback system for user suggestions
* Firebase-based data storage for purchases

## Technologies Used

Frontend:
HTML
CSS
JavaScript

Backend / Database:
Firebase

Form Handling:
Formspree API

## Purpose of the Project

The goal of this project was to create a simple platform where beginners can access programming learning materials and reference books in a single place.
